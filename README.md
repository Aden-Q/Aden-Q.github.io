# Aden-Q.github.io
Powered by `Hexo`, `GitHub`, `PicGo`, and `Node.js`

## Usage

### Create a new post
```bash
$ hexo new "title"
```

### Create a new draft
```bash
$ hexo new draft "title"
```

### Publish a draft
```bash
$ hexo publish "title"
```

### Generate static files
```bash
$ hexo generate
```

### Start a local server
```bash
$ hexo server
```
