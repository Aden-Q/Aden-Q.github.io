# Aden-Q.github.io
Powered by `Hexo`, `GitHub`, `PicGo`, and `Node.js`

## Quick Start

### Create a new post
```bash
$ hexo new "new post"
```

### Create a new draft
```bash
$ hexo new "new draft"
```

### Publish a draft
```bash
$ hexo publish "new draft"
```

### Generate static files
```bash
$ hexo generate
```

### Start a local server
```bash
$ hexo server
```
